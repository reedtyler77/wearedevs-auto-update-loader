## Overview

WeAreDevs Executor is a Windows application built for loading and executing Lua scripts inside Roblox sessions. The interface is intentionally simple: you can paste in your own scripts or open the integrated hub to access ready-made automation scripts. Its one-click injection flow is meant to connect to Roblox processes without requiring manual setup.

## FAQ

**Is WeAreDevs Executor safe to use?**  
As with any third-party tool that interacts with Roblox, you use it at your own risk. The software is offered as-is, with no promise of detection resistance or account safety.

**Will it work with the latest Roblox update?**  
The auto-update engine is intended to keep up with current Roblox versions. If injection does not work, look for a newer release.

**Do I need a key or subscription?**  
No. WeAreDevs Executor is entirely free and does not ask for a key, license, or payment.

**Can my Roblox account get banned?**  
Using script executors goes against Roblox's Terms of Service. Account action is always possible. If that is a concern, use an alternate account.

**Where are my scripts stored?**  
Local scripts are stored in `scripts/local/`. The queue database lives in `data/queue.db` and uses SQLite format.
